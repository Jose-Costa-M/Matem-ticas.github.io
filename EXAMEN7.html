1.-Se define como una función cuyo dominio es el conjunto de los enteros positivos
a. Una Serie.

2.-Cuál sería el término n-ésimo de la sucesión mostrada suponiendo que continua el patrón de los primeros términos.
a. an = (-1)n n/(n+1)2.
b. an = n/(n+1)2.
c. an = (-1)n n/n2.
d. an = (-1)n+1 n/(n+1)2.

a. an+1 < an para todo n positivo.
b. an+1 = an para todo n positivo.

a. ∞

a. Hiperarmónica
b. Geométrica.

d. Criterio de la razón.

a. Divergente.
b. Absolutamente convergente.
